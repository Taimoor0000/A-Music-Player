# 🎵 Web Music Player

A smooth and interactive **web-based music player** built with **HTML, CSS, and JavaScript**.

This project includes playlist handling, search, category filtering, and core playback controls.

---

## ✨ Features

- ▶️ Play music  
- ⏸ Pause music  
- ⏭ Next / skip track  
- ⏮ Previous track  
- 🔊 Volume control  
- 📃 Playlist display  
- 🔎 Search by song title or artist  
- 🗂 Filter songs by category  
- ⏱ Live progress bar + seek control  
- 💜 Interactive dark-purple UI with smooth animations

---

## 🧱 Tech Stack

- **HTML5** (`music.html`)
- **CSS3** (`music.css`)
- **Vanilla JavaScript** (`music.js`)

## 🚀 How to Run

1. Download or clone this project
2. Open the folder
3. Double-click `music.html`

**Recommended:** Use VS Code + Live Server for smoother development/testing.

---

## 🎮 How to Use

1. Choose a song from the playlist (or press Play)
2. Use:
   - **Play/Pause** for playback control
   - **Prev/Next** to skip tracks
   - **Volume slider** to adjust sound
   - **Progress bar** to seek within a track
3. Use search box to find songs quickly
4. Use category dropdown to filter the playlist

---

## 📌 Notes

- This version uses sample audio URLs for demonstration.
- Internet connection may be required for externally hosted tracks.
- You can replace URLs in `music.js` with your own local or hosted audio files.
